# Internship Level1 Task 3

This repository contains a simple web project with the following files:

- `index.html` - The main HTML page.
- `style.css` - Styles for the webpage.
- `script.js` - JavaScript for interactive behavior.

## Overview

The project is part of an internship assignment focusing on building a basic web page using HTML, CSS, and JavaScript. It demonstrates structuring a webpage, applying styles, and adding scripting for functionality.

## Usage

1. Open `index.html` in a web browser to view the page.
2. Edit `style.css` to change the appearance.
3. Modify `script.js` to add or update interactive features.
